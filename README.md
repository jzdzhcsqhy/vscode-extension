# Cunami

Cunami 是一个为程序员提供的工具，旨在帮助用户优化、格式化和生成代码。它支持多种功能，使得编码过程更加高效。

## 特性

- **优化代码**：通过提供代码片段，Cunami 会帮助您优化代码。
- **格式化代码**：按照指定的风格格式化代码，默认支持 C++ 风格。
- **添加注释**：为代码自动添加注释，帮助提高代码可读性。
- **代码生成**：根据需求生成函数代码，并提供详细的注释。
